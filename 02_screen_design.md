# 02. 画面設計

## 1. 画面一覧

| 画面ID | 画面名 | Activity案 |
|---|---|---|
| SCR-01 | ホーム画面 | MainActivity |
| SCR-02 | カメラ撮影画面 | CameraActivity |
| SCR-03 | 撮影後顧客選択画面 | PhotoCustomerSelectActivity |
| SCR-04 | 顧客登録画面 | CustomerRegisterActivity |
| SCR-05 | 顧客一覧画面 | CustomerListActivity |
| SCR-06 | 顧客詳細画面 | CustomerDetailActivity |
| SCR-07 | 写真詳細画面 | PhotoDetailActivity |

## 2. ホーム画面

### 表示項目

- アプリ名：おかんのカルテ
- カメラボタン
- 顧客登録ボタン
- 顧客一覧ボタン

### レイアウト案

```text
+----------------------------+
| おかんのカルテ              |
+----------------------------+
|                            |
|        [ カメラ ]           |
|                            |
|        [ 顧客登録 ]         |
|                            |
|        [ 顧客一覧 ]         |
|                            |
+----------------------------+
```

## 3. 顧客登録画面

### 入力項目

| 項目 | 必須 | 内容 |
|---|---|---|
| 顧客名 | 必須 | 顧客の氏名 |
| フリガナ | 任意 | 検索用 |
| 電話番号 | 任意 | 電話番号 |
| 郵便番号 | 任意 | 例：460-0000 |
| 住所 | 任意 | 顧客住所 |
| メモ | 任意 | 髪質、希望、注意事項など |

### ボタン

- 登録する
- 戻る

## 4. 顧客一覧画面

### 表示項目

- 検索欄
- 顧客一覧
- 顧客名
- フリガナ
- 電話番号
- 最新撮影日
- 写真枚数

### 並び順

最終撮影日が新しい順。

### 検索対象

- 顧客名
- フリガナ
- 電話番号

## 5. 顧客詳細画面

### 表示項目

- 顧客名
- フリガナ
- 電話番号
- 郵便番号
- 住所
- メモ
- 写真グリッド

### 写真表示

2〜3列のグリッド表示。

## 6. カメラ撮影画面

### 表示項目

- 撮影ボタン
- 戻るボタン
- 案内テキスト

### 案内文

```text
撮影後に顧客を選択します
```

## 7. 撮影後顧客選択画面

### 表示項目

- 撮影写真プレビュー
- 顧客検索欄
- 顧客一覧
- 新規顧客登録ボタン
- この顧客に紐づけるボタン
- キャンセルボタン

## 8. 写真詳細画面

### 表示項目

- 写真
- 顧客名
- 撮影日
- メモ
- 削除ボタン
