# Calendar Feature Prompt

## 目的

来店履歴をカレンダーから確認できるようにする。

## 基本方針

- DB構造はできるだけ変更しない。
- `Photo.takenDate` を活用する。
- 日付別に写真、顧客、来店履歴を取得する。
- 顧客詳細画面への遷移は `"customerId"` extraを使う。
- Stepごとに進め、各Step完了後はレビュー待ちにする。

## 確認項目

- 日付一覧が取得できる。
- カレンダーから日付を選べる。
- 選択日の来店顧客が表示される。
- 顧客タップで詳細へ遷移する。
- 写真保存、顧客一覧、CSV出力を壊さない。
