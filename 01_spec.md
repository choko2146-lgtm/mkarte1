# 01. おかんのカルテ アプリ仕様書

## 1. アプリ名

おかんのカルテ

## 2. 目的

美容室・個人サロンで、顧客情報と施術写真を簡単に管理するためのAndroidアプリを作成する。

撮影した写真を端末内に保存し、撮影後に該当する顧客を選択して、写真に以下の情報を紐づける。

- 顧客名
- 撮影日
- 写真メモ
- 保存ファイル名
- 保存Uri

## 3. 開発環境

| 項目 | 内容 |
|---|---|
| IDE | Android Studio |
| 補助開発環境 | VS Code + Codex |
| 言語 | Java |
| UI | XMLレイアウト |
| DB | Room Database |
| 写真保存 | 端末内保存 |
| 保存先 | Pictures/おかんのカルテ/ |

## 4. 初期版の対象機能

- ホーム画面
- カメラ撮影
- 撮影後の顧客選択
- 撮影後の新規顧客登録
- 顧客登録
- 顧客一覧
- 顧客詳細
- 写真詳細
- 写真メモ
- 写真削除
- 顧客削除
- Room Database保存
- 端末内写真保存

## 5. 初期版の対象外

- クラウド同期
- ログイン
- 予約管理
- 売上管理
- 複数端末共有
- PDF出力
- 動画撮影
- 郵便番号から住所自動入力
- 写真への手書きメモ
- アプリロック

## 6. 基本仕様

アプリ起動時には、以下の3つのボタンを表示する。

- カメラ
- 顧客登録
- 顧客一覧

撮影は以下の流れとする。

1. カメラボタンを押す
2. 撮影画面を表示する
3. 撮影ボタンを押す
4. カメラで写真を撮る
5. 写真を一時保存する
6. 顧客選択画面を表示する
7. 顧客を選択する
8. 顧客が未登録なら新規登録する
9. 写真と顧客を紐づける
10. 正式ファイル名で保存する
11. Room Databaseに写真情報を登録する

## 7. UI方針

初期版は白ベースのシンプルUIとする。

- 白背景
- 大きめのボタン
- 見やすい文字サイズ
- 美容室向けの清潔感
- 実装しやすい構成

## 8. 受入条件

以下を満たせば初期版完成とする。

1. 起動時に「カメラ」「顧客登録」「顧客一覧」が表示される
2. 顧客登録ができる
3. 郵便番号・住所を登録できる
4. 顧客一覧を最終撮影日順に表示できる
5. 顧客名・フリガナ・電話番号で検索できる
6. 写真撮影ができる
7. 撮影後に顧客を選択できる
8. 未登録顧客を撮影後に登録できる
9. 写真と顧客を紐づけできる
10. 顧客詳細で写真をグリッド表示できる
11. 写真削除時にDBと端末画像を削除できる
12. 顧客削除時に関連写真も削除できる
